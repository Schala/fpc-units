Description

Fields

id: string[5];

The header show ever the same string: PATCH note that the string is not NULL terminated.

offset: dword;

The offset where the patch will be placed in the file to patch

size: word;

The size of the data to put from the specified offset in the patching file. If when you read the size value of a record this field contains 0 you have a RLE encoded patch.

rlesize: word;
 
data: pointer;

Contains a number of Size bytes of data to be copied in the file to patch

value: byte;

This is the value to write RLESize times starting from Offset

eof_pos: string[3];

A string (not NULL terminated) saying EOF

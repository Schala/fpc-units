Description

A TIM file is a standard image file format for the Sony ˜PlayStation. The file structure closely mimics the way textures are managed in the frame buffer by the GPU. TIM files are little endian-based.

Constants

MAGIC = $10;

The header starts with a 'tag' byte; this value is constant for all TIM files and must be $10.

TIM_VERSION = 0;

denotes the version of the file format. At present, only version '0' TIM files are known to exist.

BPP4 = $08;

4-bit (color indices)

8-bit (color indices)
